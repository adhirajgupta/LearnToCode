This is an HTML document

Believe it or not, this document can actually be even simpler. If you are viewing this in a browser, right-click and select "View Page Source" to see the code for this document.

HTML is a markup language - meaning that you use tags to describe the content within. For example, this paragraph is in a <p> tag. There are a bunch of different tags in HTML. For nearly all tags, the only real difference between them is default behavior. The <h1> tag used for "This is an HTML document" makes the text larger. H1 stands for "header level 1." There are pre-defined headers for level 1-5. H1, H2, H3, H4, and H5...

One quirk of HTML is that it ignores something called "whitespace" which means extra spaces beyond the first. If you look at the code for this sentence, you will see that there are tons of extra spaces.

Opening and Closing Tags

Every tag (with very few exceptions) have a beginning and ending tag. The beginning tag is the tag name, and the end has a / before it. For example, <p> is the start tag for a P and </p> would be the end. Any content between the tags would be part of that P tag.

Why do we need tags?

Tags help to logically organize content and tell the web browser how to display content. Additionally, you can use things like classes and IDs to apply different styling (see CSS tutorials).
